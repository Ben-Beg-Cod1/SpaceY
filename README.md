# SpaceY
Repository for Capstone project 

## The repository contents 
This repository contains several jupyter notebooks that collectively make up the final project. The final project is an online powerpoint presentation pertaining to the most important data-driven insights from the project, fit for presentation to stakeholders of the Space Y company. 

## Background information 

The project was a fictional data science project that makes use of several soft skills learned during the IBM course - Data Science on coursera. These include data collection methods using web scraping and API calls, data wrangling methods, SQL, visualization, predictive modelling and dashboard creation (DASH).
